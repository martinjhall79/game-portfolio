Game Dev Resources

Articles, Tutorials & Assets To Help Us Make Better Games

hi@martinhall.games

Let’s make games!

This is an archive of the best Unity and game dev resources I’ve found to help me make better games. The collection includes tutorials, assets, tips and guides. I’ve also included some official legacy Unity projects that I’m hosting here so they don’t disappear into the digital ether.

Tutorials

The best game development and Unity tutorials I’ve used on my learning journey.

Particle Sea Tutorial

Learn how to create an animated particle sea effect in Unity, with this written tutorial by Rafal Wilinski. Shows how to manipulate a particle system with code and teaches you how to use Perlin noise.

Check out the particle sea tutorial by Rafal Wilinski

League Of Legends VFX

VFX artist Kevin Leroy has posted a bunch of fan-made League of Legends ability effects, which were made in Unity, on Reddit. These are easily some of the best Unity created VFX I've seen.

Check out the VFX on Kevin Leroy's YouTube channel

A* Pathfinding

An easy to understand and follow tutorial on implementing A* pathfinding with Unity by Sebastian Lague.

Watch the A* pathfinding tutorial series on YouTube

Procedural Caves

Another amazing tutorial series by Sebastian Lague on how to create procedural caves with Unity. This series was originally on YouTube but it is so good that Unity have actually included it in their official Learn section too.

Watch the tutorials at Unity Learn

Top Down Shooter

Another great tutorial series by Sebastian Lague on how to create a top down shooter. The first parts of the series are pretty standard but the procedural map generator featured in parts 8 through 13 is really cool.

Watch the tutorials on YouTube

Procedural Maze

One of the best written tutorials I've ever followed, by Jasper Flick on creating a procedural maze. The resulting project is almost a fully playable game.

See the tutorials on Jasper Flick's website

Intro To Shaders

A really informative 5-part written tutorial series for developers with little to no experience with Unity shader coding.

See part one of the tutorials by Alan Zucconi

Learn To Draw Pixel Art

An easy to follow ten step written tutorial series on how to draw pixel art by Derek Yu. Derek is the game dev who created Spelunky, which is one of my all time favourite games, which also makes him one of my favorite game developers.

See the pixel art tutorials by Derek Yu

Assets

The most useful assets I've used like tools, models, textures and image effects that you can just drop into your Unity projects.

Subpixel Morphological Anti Aliasing

If you're using deferred shading, this is one of the best antialiasing currently available for Unity. Best of all it's completely free and open source. Created by Thomas Hourdel.

Download it from Thomas's GitHub repository
See the Unity forum thread for a discussion about the antialiasing asset
Check out Thomas's high quality assets on the Unity asset store

Screenspace Reflections

A free and open source space reflections image effect for Unity development, created by Ben Hopkins.

Get the effect from Ben's Github
Read about how the effect was implemented on Ben's blog

Supplemental Assets

Other useful assets that I'm copying from the Unity Educator Framework.

Textures

  • CG Textures (www.cgtextures.com) The world's larget texture resource site. The site offers a plethora of high quality photo references that you can use to generate textures. In addition, this site offers a small selection of useful texture creation tutorials.
  • Texture King (www.textureking.com) Offers textures in concrete, grunge, wood, stone/rock, metals, fabric, paint, rust, plastic, dirt/sand, liquids, glass and plaster.
  • 2 Textured (www.2textured.com) Good resource for weapon, food, animal, and vehicle textures and images among other things.
  • Stock Textures (www.stocktextures.com) Free and open archive for high-quality, tilealbe textures.

Audio Assets

  • Flashkit (www.flashkit.com/soundfx) List of shareware and freeware Sound FX in the following categories: ambience, communication, cartoon, creatures, domestic, electronic, industrial commercial, instruments, interfaces, mayhem, mechanical, nature, people, recreation, and transportation.
  • Indie Game Music (http://indiegamemusic.com) Offers free music for games. Even though you can use the music for free, some of the files do require you to email the creator and let them know you are going to use the file(s).
  • Jamendo (www.jamendo.com) Jamendo offers free streaming and download of music published under Creative Commons licenses. On Jamendo, artists grant use of their music for private uses free of cost.
  • Free Sound Effects (www.freesoundeffects.com) While many of the sounds on this site must be paid for, there is a good selection of free sound effect as well.

Model Assets

  • Turbo Squid (www.turbo.com/soundfx) Offers models in a multitude of formats. There are models for purchase but also a great number of free models.
  • Archive 3D (www.3dxtras.com) A large variety of models in *.3DS and *.GSM formats.
  • Artist-3d (www.artist-3d.com) A free exchange directory of royalty free 3d models in these formats: 3ds, obj, pz3, pp2, max, mb.
  • Exchange 3D (www.exchange3d.com) While Exchange3d is primarily a commercial site, they do offer numerous pages of free, high quality models.

Bug Tracking Software

  • Buzilla (www.bugzilla.org) Bugzilla is a server-side bug tracking system that allows you to keep track of outstanding bugs in your product. You can track bugs and code changes, communicate with collaborators, submit and review patches, and manage quality assurance.
  • Landfill (http://landfill.bugzilla.org/) Landfill is the home of test installations for Bugzilla. These are demo installations that you can use to ‘try out’ Bugzilla.
